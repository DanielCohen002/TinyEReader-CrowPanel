TinyEReader — CrowPanel ESP32 E-Paper HMI 2.13"

Flash the latest firmware directly from this page — no software to install. Plug the board in over USB, then click below.

Your browser doesn't support Web Serial. Open this page in Chrome or Edge on a desktop computer. This page needs to be loaded over HTTPS.
Notes
  • Works in Chrome or Edge on desktop only (Web Serial isn't supported on Firefox, Safari, or mobile browsers).
  • This erases and reflashes the whole board — any books already on it will be wiped along with your saved place. Back up anything you care about first (see the README).
  • If the port doesn't show up, try a different USB cable/port, or close any other program (Arduino IDE, Serial Monitor) that might have it open.
