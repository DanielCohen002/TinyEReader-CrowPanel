TinyEReader — CrowPanel

Flash the latest firmware directly from this page — no software to install. Plug the board in over USB, then click below.

Your browser doesn't support Web Serial. Open this page in Chrome or Edge on a desktop computer. This page needs to be loaded over HTTPS.
Notes
  • Works in Chrome or Edge on desktop only (Web Serial isn't supported on Firefox, Safari, or mobile browsers).
  • You may be asked whether to erase the board first — decline it for a normal update. Declining only overwrites the firmware itself, leaving your books, bookmarks, and settings untouched; checking it wipes everything, books included. Only check it if you actually want a clean slate, or you're flashing this for the first time onto a board that had different, incompatible firmware on it before (see README).
  • If the port doesn't show up, try a different USB cable/port, or close any other program (Arduino IDE, Serial Monitor) that might have it open.

Converting an EPUB? The device's own upload page does this automatically, but there's also a standalone EPUB → TXT converter if you'd rather convert ahead of time.
